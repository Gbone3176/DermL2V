# CLS Text Scripts

## Layout

- `configs/baselines/`: baseline CLS JSON configs.
- `configs/derml2v/`: DermL2V CLS JSON configs.
- `configs/llm2vec/`: LLM2Vec base / cp0 CLS JSON configs.
- `configs/offline_embeddings/`: offline-embedding CLS configs.
- `launch/`: shell launch examples and grouped offline embedding launchers.
- `pipelines/`: DermL2V 3.2-1.3B CLS pipeline drivers.

New configs should go under the most specific `configs/` subdirectory instead of the directory root.
